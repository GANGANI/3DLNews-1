* Do you know what the CAA record is?

Suggested article: What is CAA record?

The CAA record (Certification Authority Authorization) is a DNS Resource Record. It allows a DNS domain name holder to specify one or more Certification Authorities (CAs) authorized to issue certificates for that domain.

How to add a CAA record?

Each CAA resource record consists of the following components:

  • FLAG
  • TYPE
  • VALUE

CAA records can set policy for the entire domain or for specific hostnames. CAA records are also inherited by subdomains. To make it clear, we can consider the following example: if a CAA record is set on example.com, it also applies to any subdomain like shop.example.com.
