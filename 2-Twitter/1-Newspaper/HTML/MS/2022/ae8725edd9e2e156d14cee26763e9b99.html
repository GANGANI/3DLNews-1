# Tips for using NLP keywords

The robot icon will generate a mini paragraph containing the term.

Follow the below steps for optimal results:

  • Place your cursor where you want the AI to generate the paragraph including the term
  • Click the Robot icon next to that term
  • Check if you are satisfied with the output otherwise generate again

Note: When placing the cursor in the editor and clicking the Robot icon, the AI considers the header above the cursor when generating the paragraph.

# Tips for using NLP keywords inside outline section

Generating a rich keyword paragraph based on the selected keywords

  • Add any keyword/keywords you want if any
  • Select the keywords you want to be included in the generated paragraph
  • Click paraphrase
  • Review the output and generate again if needed
