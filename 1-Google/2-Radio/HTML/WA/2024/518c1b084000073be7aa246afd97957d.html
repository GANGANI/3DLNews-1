            Ahh– so we commit to having the PRX episode uploaded by Monday (“Monday 23:59 Pacific Time”), because that is where many of the other public radio stations download the file from. We of course almost always have it ready at some point on Sunday, because we never try to put it off to the last minute, since most work stations use a work flow and expect a new episode by Monday evening.

            We add some extra metadata and do some extra quality checks on that upload because if we mess up, it would impact the 40 other stations that run it (who run it the week following the Sunday that it airs on C89.5). If we mess up on something minor on the version that runs on C89.5, it’s much easier to fix that, because it’s just one station. If we mess up on something that goes out to the other stations, and it went out to 40 stations, we’d have to reach out individually to those 40 stations and get them the correct files.
